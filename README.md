# dotfiles

[![Build Status][travis-svg]][travis-link]

## Installation

```sh
git clone --depth 1 https://github.com/j-/dotfiles && \
dotfiles/scripts/install.sh
```

Alternatively, with [npm][npm-site]:

```sh
npm install --global j-/dotfiles
```

## Features

* Handy aliases
* Useful functions
* Pretty command prompt
